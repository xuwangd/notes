#### 计算机、冯诺依曼式计算机、

图灵是计算机之父 也是 人工智能之父
冯诺依曼是计算机创始人（五大电子设备：输入 输出 控制 存储 运算）

- 计算机
    - 台式电脑
    - 笔记本电脑
- 冯诺依曼式计算机
    - 手机
    - 平板

- 电子设备
    - 输入器：网口、麦克风、键盘、鼠标
    - 输出器：网口、耳机、显示器
    - 运算器：CPU（适用逻辑运算、整型运算）、GPU（浮点型运算、精准运算）
    - 存储器：内存（读写快、断电清空数据）、硬盘辅存（读写较慢、持久化）
    - 控制器：主板上的一些器件
